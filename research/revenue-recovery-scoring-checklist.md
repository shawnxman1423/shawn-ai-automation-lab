# AI revenue recovery scoring checklist

Use this checklist to evaluate whether a vertical is a good fit for an AI recovery agent.

| Criterion | Strong signal | Weak signal |
|---|---|---|
| Dormant asset | thousands of leads/patients/accounts | small ad hoc list |
| Value per conversion | $300+ appointment or high-ticket case | low-margin one-off |
| Follow-up gap | staff admits no time / CRM stale | already fully automated |
| Natural trigger | renewal, recall, seasonality, quote expiry | vague “maybe interested” |
| Channel fit | SMS, WhatsApp, voice, email expected | customers dislike outreach |
| Agent scope | scheduling, intake, reminders | complex professional judgment |
| Compliance path | clear consent + audit logs | ambiguous consent/licensing |

## MVP shape

1. Import or sync dormant leads.
2. Segment by age/source/intent/value.
3. Run compliant outreach.
4. Qualify replies.
5. Book into calendar or hand off to human.
6. Write CRM notes and attribution.
7. Report recovered revenue.

## Early success metrics

- response rate
- qualified reply rate
- booked appointment rate
- show rate
- recovered revenue
- opt-out / complaint rate
- human takeover rate
